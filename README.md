# Echo-Noter 🧠✨

> Turn any YouTube video into smart, structured AI notes instantly.

Echo-Noter is a next-generation AI learning assistant that saves you hours of time by automatically converting educational YouTube videos into high-quality study materials. Just paste a video URL, and our AI extracts the transcript, simplifies complex topics, and generates bullet-point summaries, flashcards, and structured notes.

## 🚀 Features
- **Smart Note Generation**: Automatically creates well-structured, topic-wise study notes.
- **AI Concept Simplification**: Breaks down difficult technical topics into easy-to-understand explanations.
- **Instant Flashcards & Quizzes**: Automatically generates Q&A flashcards for rapid revision.
- **Modern UI**: Built with a stunning, futuristic glassmorphism design and smooth animations.

## 🛠 Tech Stack
- **Frontend**: Next.js, React, TypeScript, Tailwind CSS v4, Framer Motion
- **Backend**: Node.js, Express.js, TypeScript
- **Database**: MongoDB
- **AI/Processing**: YouTube Transcript API, OpenAI/Gemini

## 🏁 Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/Techside-Pragyan/Echo-Noter.git
cd Echo-Noter
```

### 2. Run the Frontend
```bash
cd frontend
npm install
npm run dev
```
The frontend will start at `http://localhost:3000`.

### 3. Run the Backend
```bash
cd backend
npm install
npm run dev
```
The backend will start at `http://localhost:5000`.
